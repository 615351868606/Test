## 概述

我的这本新书《Office 365开发入门指南》自2018年9月份正式上市发行以来，总体来说收到了较好的反响。此书在写作和出版过程中，Office 365这个平台也仍然在不断地改进和升级，所以我特意录制了这套培训视频，并且在网易云课堂中发行，希望对大家有所帮助。

这个库将保存课程中所提到的资源，范例代码等。
